"What Did I Just Do, and What's Next?"

You just configured and authorized Datatap to access one of your external data resources, allowing the programs to talk to each other. The next step is either:

  • Repeat the process for other datasources you want to connect.
  • Apply Schema Mapping to your Data Extract to convert/manipulate fields.
  • Learn to set a schedule for further Fetches.
  • Export your dataset to Adverity Explore.
  • Create an  External Destination to which you can send your Data Extracts
